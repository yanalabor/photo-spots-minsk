О проекте «Фото-места Минска»

«Фото-места Минска» — это интерактивная веб-платформа, предназначенная для поиска, публикации и оценки интересных локаций для фотосъемки в городе Минске. Проект ориентирован на фотографов, блогеров, путешественников и всех пользователей, заинтересованных в поиске красивых мест города.

Актуальность проекта

Обоснование необходимости разработки системы

В настоящее время фотографы, блогеры, туристы и любители городской фотографии активно используют интернет-ресурсы для поиска интересных мест для съемок. Однако специализированной платформы, объединяющей фотолокации города Минска, практически не существует.

Проект «Фото-места Минска» решает данную проблему, предоставляя пользователям возможность находить, оценивать, сохранять и самостоятельно публиковать фотолокации, а также обмениваться опытом через систему отзывов и рейтингов.

Техническая реализация проекта

Архитектура системы построена по модели «клиент — сервер — база данных»

💻 Фронтенд (клиентская часть)

При разработке пользовательского интерфейса мы стремились создать современное, адаптивное и удобное веб-приложение.

  • Интерактивная карта: реализована на основе OpenStreetMap и библиотеки Leaflet.js.
  • Кластеризация: позволяет удобно отображать большое количество объектов на карте.
  • Динамический интерфейс: взаимодействие с сервером происходит через JavaScript и REST API без перезагрузки страниц.
  • Личный кабинет: реализованы функции управления профилем, избранными местами, отзывами и собственными локациями.
  • Адаптивный дизайн: интерфейс корректно отображается на мобильных устройствах и компьютерах.

⚙️ Бэкенд (серверная часть)

Серверная часть отвечает за обработку запросов, безопасность и хранение данных.

  • REST API: сервер реализован на Python с использованием FastAPI.
  • База данных: используется реляционная СУБД MySQL для хранения пользователей, мест, отзывов и избранного.
  • Хранение изображений: реализовано локальное файловое хранилище сервера.
  • Безопасность: пароли пользователей защищены алгоритмом bcrypt.
  • Восстановление доступа: реализована система входа по одноразовому коду.
  • Модерация контента: реализована автоматическая фильтрация нецензурной лексики в пользовательских комментариях.

🏗 Архитектура системы

  • Frontend: HTML, CSS, JavaScript;
  • Backend: Python + FastAPI;
  • База данных: MySQL;
  • Картографическая система: OpenStreetMap + Leaflet.js;
  • Хранение файлов: локальное файловое хранилище.

📈 Выводы

В ходе выполнения проекта нами была разработана полноценная веб-платформа для поиска и публикации фотолокаций города Минска.

Были реализованы механизмы регистрации и авторизации пользователей, интерактивная карта, система отзывов, рейтингов, избранного, хранения изображений, восстановления доступа и фильтрации пользовательского контента.

Разработанная система является функционально завершенной и обладает возможностью дальнейшего расширения и масштабирования.
